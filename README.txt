Pasos para montar la red social en local

1º - Descomprimir el zip

2º - Dentro encontrará dos carpetas la parte de la api y la de cliente

3º - Ejecutar el servidor mongod.exe y dentro de la carpeta api npm install y npm start

4º - Correr el cliente haciendo npm install y ng serve dentro de la carpeta client

El backend se ejecuta en el puerto 3800, el frontend en el 4200

Para ver la documentación del proyecto ejecutar en la carpeta client 

npm run docs - Para generar la documentación

npm run serve-docs - Para ver la documentación (se ejecuta en el puerto 8080)
